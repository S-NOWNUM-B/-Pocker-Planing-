## Актуальная спецификация

Источником истины для endpoint является runtime-документация backend:

<div align="center">

| **Интерфейс** | **URL** |
|:---|:---|
| Swagger UI | `http://localhost:8081/swagger-ui.html` |
| OpenAPI JSON | `http://localhost:8081/v3/api-docs` |

</div>

Если есть расхождения между этим файлом и OpenAPI, приоритет у OpenAPI.

## Основные группы эндпоинтов

<div align="center">

| **Группа** | **Назначение** |
|:---|:---|
| `libraries` | Каталог библиотек, поиск, статистика и health-метрики |
| `subscriptions` | Подписки пользователей и уведомления |
| `library-updates` | Лента обновлений библиотек |
| `scan-jobs` | Постановка и мониторинг задач сканирования |
| `sources` | Список/метаданные подключенных источников |
| `licenses` | Лицензии и матрица совместимости |
| `collector` | Ручной запуск и диагностика сбора данных |
| `auth` и `auth/password-reset` | Аутентификация и восстановление доступа |
| `health` | Технические проверки сервиса |

</div>

## Административные маршруты

В проекте используются два административных префикса (историческая совместимость):

<div align="center">

| **Префикс** | **Назначение** |
|:---|:---|
| `/api/v1/admin/*` | Пользователи и статистика обслуживания |
| `/api/admin/*` | Библиотеки и скрейперы |

</div>

## Контракты ответов и ошибок

<div align="center">

| **Ситуация** | **HTTP-код** |
|:---|:---|
| Успешный ответ | `2xx` + JSON |
| Бизнес-ошибка | `4xx` |
| Системная ошибка | `5xx` |
| Пагинация | `content`, `totalElements`, `totalPages`, `currentPage` |

</div>

Пример тела ошибки:

```json
{
  "timestamp": "2026-03-13T10:00:00Z",
  "status": 400,
  "error": "Bad Request",
  "message": "Validation failed",
  "path": "/api/v1/packages/search"
}
```
